Сервис_ИзготовлениеПрототиповОборудования {

    В рамках сервиса выполянетс изготовление прототипов оборудвания для внутренних заказчиков Отдел_Производство.
    Заказчики сервиса:
    - Отдел_РазработкиОборудования
    - Отдел_СпоровожденияПроизводства

    В рамках сервиса выделяются следующие стадии:
    - Стадия_СборЗапросов
    - Стадия_ПланированиеПроизводства
    - Стадия_ИзготовлениеОборудования
    - Стадия_ВыдачаОборудования


    Стадия_СборЗапросов {

        Отдел_Производство планирует работы недельными циклами. Чтобы заказать изготовление оборудования 
        необходимо до 14:00 четверга подать Объект_ЗаявкаНаИзготовление через Технология_Redmine.

        Выход:
        Объект_ЗаявкаНаИзготовление
    }

    Стадия_ПланированиеПроизводства {

        Роль_Планировщик оценивает сроки изготовления описанного в Объект_ЗаявкаНаИзготовление.
        При необходимости Роль_Планировщик привлекает для оценик трудоемкости и возможности изготовления Роль_МонтажникПечатныхПлат,
        Роль_ИнженерФрезеровщик. 
        Если задачу невозможно включить в план из-за отсутствия технической возможности Роль_Планировщик уведомляет об этом заказчика.
        Роль_Планировщик публикует Объект_НедельныйПланПроизводства публикуется каждую пятницу в 12:00

        Вход:
        - Объект_ЗаявкаНаИзготовление

        Выход:
        - Объект_НедельныйПланПроизводства

    }
